## 🎯 Швидкі поради

### Як запустити проект

**Способ 1 (Найпростіший - Windows):**
1. Подвійно клікніть на `start.bat` файл
2. Дочекайтеся інсталяції залежностей
3. Браузер відкриється автоматично на `http://localhost:3000`

**Способ 2 (PowerShell - Windows):**
```powershell
.\start.ps1
```

**Способ 3 (Терміналь):**
```bash
npm install
npm run scrape
npm start
```

### 🔄 Оновлення контенту

Щоб завантажити найновіший контент з Telegraph та Telegram:
```bash
npm run scrape
```

Це може зайняти 1-2 хвилини. Дані збережу в `data/chapters.json`.

### 🛠️ Налаштування

Всі налаштування в CSS змінних (`public/styles.css`):
```css
--dark-bg: #1a1a1a      /* Основний темний фон */
--accent: #b8b8b8       /* Сірий акцент */
--gray-200: #e0e0e0     /* Основний текст */
```

Змініть їх для vlastní гами кольорів!

### 🐛 Проблеми?

| Проблема | Рішення |
|----------|---------|
| "npm не знайдено" | Встановіть Node.js з nodejs.org |
| Порт 3000 зайнятий | Змініть PORT в `server.js` |
| Скрейпер не працює | Перевірте інтернет, Telegraph може блокувати |
| Сторінка не завантажується | Перезавантажте `Ctrl+Shift+R` |
